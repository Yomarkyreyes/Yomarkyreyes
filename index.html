Historia y Evolución de HTML

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Fue desarrollado por Tim Berners-Lee en 1991.

Evolución de HTML

Desde su creación, HTML ha pasado por varias versiones importantes que han ampliado sus capacidades y mejorado su funcionalidad.

Versiones Principales de HTML

  • HTML 1.0 (1991)
  • HTML 2.0 (1995)
  • HTML 3.2 (1997)
  • HTML 4.01 (1999)
  • HTML 5 (BORRADOR) (2008)
  • HTML5 (2014)

Características Destacadas de HTML5

  1. Soporte de multimedia nativa (audio y video).
  2. Elementos semánticos nuevos como <article>, <section> y <nav>.
  3. Mejor soporte para gráficos y juegos (canvas y SVG).
  4. Almacenamiento web local.
